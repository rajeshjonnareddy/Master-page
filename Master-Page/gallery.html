Rover

It’s a smart Robot

This is one of my greatest project. It's named after mars rover, but this rover uses a micro processor called Raspberry Pi 3. Powered by 3000mAh lithium polymer battery and a power-bank.

This Rover is platform independent because it can be controlled by any kind of web browser which supports scripts. And it is one the best smart feature is, it can go to autopilot or you can say autonomous mode by just one click of autopilot mode button. And it can also be controlled with keyboard game control by simply clicking Keyboard Mode Button. This Rover can be accessed from different places with live FPV streaming because the rover is based on cloud computing, it accesses internet. It means the rover can be accessed from anywhere. Suppose the rover is in U.S and the user is in India, it can be easily accessed and controlled from India through internet.

by Hemaal Taras Hansda, 12/11/2017

Know more
×

The full control of the Rover runs on a lighttpd Webserver. The User Interface (UI) is made up of javascript and HTML. The whole things is running on Raspbian OS. To program the GPIO's for different works and features, different programming languages are used. Like python for Autonomous mode and Keyboard mode and CGI scripts are used both for interacting with the camera servos and rover Manual Control. C programming language is also used for setting up the servos.
